# Адресная книга

* Приложение адресная книга - хранение информации о контакте: его адреса, имя и номера телефонов.
* Данные о контактах хранятся в базе данных.
* Управление данными о пользователях (добавление новых пользователей, изменение и удаление информации о пользователе) производится посредством REST WebService. 
* Используемые технологии: SpringBoot, SpringMVC, SpringData, Map Struct, Lombok, Hibernate, Heroku, PostgreSQL. Для сборки проекта использовался Maven.

## Конфигурация приложения

Скрипты создания и наполнения таблиц находятся в папке resources

### Управление данными

Управление данными о пользователях происходит посредством HTTP запросов на localhost через порт 8081.

Список всех пользователей:

GET запрос на http://localhost:8081/address-book/persons

Информация о пользователе по id:

GET запрос на http://localhost:8081/address-book/persons/{id}

Информация о пользователе по имени:

GET запрос на http://localhost:8081/address-book/persons/?name=NameToFind

Создание нового блока данных:

POST запрос на http://localhost:8081/address-book/persons

Тело запроса:

JSON с параметрами name, contacts и addresses:
```
{
    "name": "Person_name",
    "contacts": [
        {
            "telephone": "+375291111444"
        },
        {
            "telephone": "+375291111111"
        },
        {
            "telephone": "+375291111111"
        }
    ],
    "addresses": [
        {
            "city": "Minsk",
            "street": "Esenina"
        },
        {
            "city": "Minsk",
            "street": "Lenina"
        }
    ]
}
```

Изменение блока данных:

PUT запрос на http://localhost:8081/address-book/persons

Тело запроса:

такое же как и при создании + поле id:.

```
{
    "id": 13,
    "name": "Person_name",
    "contacts": [
        {
            "telephone": "+375291111555"
        }
    ],
    "addresses": [
        {
            "city": "Minsk",
            "street": "Esenina"
        }
    ]
}
```

Удаление информации о пользователе по id:

DELETE запрос на http://localhost:8081/address-book/persons/{id}
